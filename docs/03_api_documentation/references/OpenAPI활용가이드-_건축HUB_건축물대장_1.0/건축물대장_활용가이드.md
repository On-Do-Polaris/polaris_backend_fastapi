   flrNo 층번호 침수깊이별 피해층 계산 핵심. 침수 1m → 지하1층 전체 + 1층 일부
   area 면적(㎡) 층별 피해액 정밀 산정 = area × 단위면적당 피해액
   mainPurpsCd 주용도코드 지하층 기계실/전기실 식별 핵심
   etcPurps 기타용도 중요설비 위치 파악 ("기계실", "전기실", "저수조", "펌프실" 키워드 추출). 침수 시 건물 전체 운영 중단 리스크
   strctCd 구조코드 층별 구조 차이 파악 (고층부 철골 vs 저층부 콘크리트)
   활용 시나리오
   text
   침수 1.0m 발생 시:

- 지하1층 (flrNo=-1) 전체 침수 → area × 100% 피해
- 1층 (flrNo=1) 일부 침수 → area × 50% 피해
- 2층 이상 안전

중요설비 파악:

- etcPurps에서 "기계실", "전기실" 검색 → 침수 시 전체 운영 중단
