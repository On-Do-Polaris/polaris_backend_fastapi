## 4. getBrFlrOulnInfo (건축물대장 층별개요 조회) ⭐ 최우선 핵심

### 필수 항목 (전체)

| 항목명(영문)    | 항목명(국문) | TCFD 활용 목적                                                                                                  |
| :-------------- | :----------- | :-------------------------------------------------------------------------------------------------------------- |
| **flrGbCd**     | 층구분코드   | 지상/지하 구분 (침수 리스크 차별화)                                                                             |
| **flrNo**       | 층번호       | **침수깊이별 피해층 계산 핵심. 침수 1m → 지하1층 전체 + 1층 일부**                                              |
| **area**        | 면적(㎡)     | **층별 피해액 정밀 산정 = area × 단위면적당 피해액**                                                            |
| **mainPurpsCd** | 주용도코드   | **지하층 기계실/전기실 식별 핵심**                                                                              |
| **etcPurps**    | 기타용도     | **중요설비 위치 파악 ("기계실", "전기실", "저수조", "펌프실" 키워드 추출). 침수 시 건물 전체 운영 중단 리스크** |

### 활용 시나리오

```
침수 1.0m 발생 시:
- 지하1층 (flrNo=-1) 전체 침수 → area × 100% 피해
- 1층 (flrNo=1) 일부 침수 → area × 50% 피해
- 2층 이상 안전

중요설비 파악:
- etcPurps에서 "기계실", "전기실" 검색 → 침수 시 전체 운영 중단
- etcPurps에서 "저수조", "펌프실" 검색 → 급수 시스템 마비
```

---

### 활용 시나리오

```
자산가치 손실 산정:
100년 빈도 홍수 → 주택가격 10% 하락
→ 손실액 = hsprc × 0.1 × 발생확률(0.01)

포트폴리오 집중도:
고위험 지역 자산 총액 = Σ(hsprc) where 침수위험지역
→ ESG 공시: "기후 리스크 노출 자산 XX억원"
```

## 데이터 수집 워크플로우

### 1단계: 기본개요 조회로 식별자 수집

```
getBrBasisOulnInfo(sigunguCd, bjdongCd, bun, ji)
→ 해당 지번의 모든 건축물(동) 식별
```

### 2단계: 각 건축물별 상세 정보 수집

```
  - getBrRecapTitleInfo → 단지 전체 규모
  - getBrTitleInfo → 구조, 층수, 내진능력
  - getBrFlrOulnInfo → 층별 면적/용도 (침수 분석 핵심)
  - getBrHsprcInfo → 자산가치 (주택만 해당)
```

### 3단계: TCFD 분석

```
물리적 리스크:
- 위치(platPlc) + 기후 시나리오 → 침수깊이 산정
- 침수깊이 + 층별정보(flrNo, area) → 층별 피해액 적산
- 구조(strctCd) + 내진능력(rserthqkAblty) → 지진 피해율

```

## 핵심 요약

### 절대 필수 항목 (Top 5)

1. **mgmBldrgstPk / bldgId** - 건축물 고유 식별자
2. **platPlc / sigunguCd / bjdongCd** - 위치 정보 (기후 리스크 매핑)
3. **flrNo / area / etcPurps** - 층별 상세 (침수 정밀 분석)
4. **rserthqkAblty** - 내진능력 (지진 리스크 정량화)

### 중요 항목

- **구조(strctCd), 층수(grndFlrCnt, ugrndFlrCnt)** - 건축물 취약도
- **totArea, archArea, mainPurpsCd** - 자산 규모 및 용도
- **useAprDay** - 건축연도 (노후도 분석)
- **engrGrade, gnBldGrade** - 에너지/친환경 등급 (전환 리스크)

이 항목들만으로 **TCFD 물리적 리스크 정량화 및 Scope 3 배출량 추정**이 충분히 가능합니다.[^1]
